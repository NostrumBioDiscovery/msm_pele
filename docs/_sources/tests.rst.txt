Run production test
-----------------------


  It is recommended to run the production tests after installation by:

1- **Extract test data**::

  tar -xvf msm_pele/test/data.tar.gz

2- **Run test**::

  cd /path/where/you/want/to/runTheTest/from/

  sbatch msm_pele/test/test.sl
